COMMAND LINE ARGUMENTS

 

-An                                   This is used to run two or more copies of prime95 from the same directory.  Using this command line argument causes Prime95 to use a different set of filenames for the INI files, the results file, the log file, and the spool file.  Just use a different value of n for each copy of prime95.exe you start.

-t                                        Run the torture test.  Same as Options/Torture Test.

-Wdirectory               This tells Prime95 to find all its files in a different directory than the executable.
