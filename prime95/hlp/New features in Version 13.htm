New features in Version 13.2 of prime95.exe

 

1.        Minor adjustments made to the per-iteration error checking code.  It was occasionally raising a FATAL ERROR when it should not have been.

 

New features in Version 13.1 of prime95.exe

 

1.        Faster!  Prime now supports a wider variety of FFT run lengths.  This means Lucas-Lehmer tests on exponents between 1.33M and 2.33M will be 13 to 40% faster.
